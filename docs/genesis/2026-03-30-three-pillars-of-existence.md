## Implications

### For AX-8 (Constructed Polis)

The polis must represent all three orders of existence. The current requirement is "minimum 3 lenses per Triadic Review Protocol." The three pillars refine this:

**Every polis must include at minimum:**
1. A **natural science lens** — examines the formation's physical properties (performance, resource consumption, signal flow rates, metabolic health, entropy resistance)
2. A **social science lens** — examines the formation's institutional properties (reception, community engagement, economic viability, cultural identity, labor model)
3. A **formal science lens** — examines the formation's structural properties (correctness, completeness, logical consistency, type safety, algorithmic efficiency)

This is not arbitrary diversity. These are the three orders of existence. Miss one and the polis has a blind spot the size of a pillar of reality.

### For the Allocative Institution (the critical gap)

Economics is the only fundamental discipline with NO constitutional expression. This means:

1. The system has no theory of **value** — what makes one formation's output more valuable than another's?
2. The system has no **budget governance** — how is finite human attention allocated across 127 formations?
3. The system has no **pricing framework** — how is consulting work valued? How are grant applications prioritized?
4. The system has no **token economy** — how is compute, API cost, and LLM token expenditure governed?
5. The system has no **carrying capacity** concept — how many living formations can one person sustain?

The Client Pillar spec (commerce--meta, approved 2026-03-28) addresses items 3 partially. But the general allocative institution — the economics of the organism — is the next constitutional obligation.
