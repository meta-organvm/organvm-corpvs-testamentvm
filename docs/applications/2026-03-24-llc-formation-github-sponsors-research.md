# LLC Formation & GitHub Sponsors Research

**Date:** 2026-03-24
**Status:** Research complete

---

## Key Findings

### LLC Formation — Fastest Path

**Recommendation:** Form in home state, not Delaware. Delaware advantages are irrelevant for single-member LLCs. You'd pay fees in TWO states (DE + home state foreign qualification).

**Timeline: 1-5 business days from zero to bank account.**

| Step | Time |
|------|------|
| File Articles of Organization online | Same day (many states offer instant/expedited) |
| Draft & sign Operating Agreement | Same day |
| Apply for EIN at irs.gov | Same day (10 min, instant result, FREE) |
| Open business bank account (Mercury/Relay/Novo) | 1-3 business days |

**Cost:** $50-500 total (home state), vs $440-1,190+ for Delaware.

**PBC conversion note:** Cannot convert LLC to PBC directly. Must convert to corporation first, then amend to PBC. If PBC is 2+ years out, start with LLC now.

**California warning:** $800 minimum annual franchise tax for LLCs regardless of income.

### GitHub Sponsors

**Fastest path:** Set up under personal account first (no entity needed). Migrate to org later.

**Approval timeline:** Days to 2-3 weeks. Having established repos speeds it.

**Fees:** 0% GitHub fee on personal sponsorships. Up to 6% on org (3% processing + 3% service, eliminable via invoice billing).

**Can link to fiscal sponsor** via Open Collective integration.

**Recommended tiers:**
- $5/mo — Community (name in SPONSORS.md)
- $15/mo — Backer (logo in README)
- $50/mo — Bronze (priority issues, updates)
- $100/mo — Silver (website logo, quarterly call)
- $500/mo — Gold (dedicated support, roadmap input)
- $1,000+/mo — Platinum (consulting hours, SLA)

**Setup:** Add `FUNDING.yml` to key repositories with sponsor links.

## Execution Sequence

**Day 1:** File LLC → sign Operating Agreement → get EIN (instant) → set up personal GitHub Sponsors
**Day 2-3:** Open business bank account → set up org GitHub Sponsors → configure Stripe Connect
**Week 2-3:** Approval → configure tiers → add FUNDING.yml to repos
